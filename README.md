# Createx SEO Agency Website

A fully responsive multi-section landing page built with **HTML** and **Tailwind CSS**.  
This project is a modern business website for a digital marketing/SEO agency, designed with clean UI, responsive layouts, and professional styling.

---

## Project Overview

The Createx SEO Agency website is designed to present a professional digital marketing company online.  
It includes multiple sections that highlight the company’s services, achievements, client testimonials, pricing plans, case studies, latest news, and contact information.

The goal of this project was to practice:
- Building complex responsive layouts
- Working with Tailwind CSS utility classes
- Structuring large landing pages
- Improving UI/UX design implementation from Figma/design reference
- Writing clean and organized frontend code
